# diabetes-decision-tree-analysis
A supervised machine learning pipeline for diabetes outcome prediction using a Decision Tree classifier, ROC-AUC analysis, and stratified k-fold cross-validation.

pandas: Data loading, inspection, and feature/target preparation
scikit-learn: Train-test split, DecisionTreeClassifier, performance metrics, ROC-AUC, StratifiedKFold, and cross-validation
matplotlib: ROC curve visualization
